## 1. 接口规范

### 1.1 请求规范

#### 1.1.1 请求头
```http
Content-Type: application/json
Authorization: Bearer <token>
User-Agent: NutriAI-Frontend/1.0
X-Request-ID: <uuid>
```

#### 1.1.2 统一响应格式

**成功响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    // 业务数据
  },
  "timestamp": 1701518400000
}
```

**失败响应**：
```json
{
  "code": 40001,
  "message": "用户名或密码错误",
  "data": null,
  "timestamp": 1701518400000
}
```

#### 1.1.3 分页响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [...], // 数据列表
    "total": 100, // 总记录数
    "page": 1, // 当前页码
    "pageSize": 10, // 每页数量
    "totalPages": 10 // 总页数
  },
  "timestamp": 1701518400000
}
```

### 1.2 HTTP状态码

| 状态码 | 说明 |
|-------|------|
| 200 | 成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未授权（未登录或token过期） |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 429 | 请求频率超限 |
| 500 | 服务器内部错误 |
| 503 | 服务不可用 |

## 2. 认证授权接口

### 2.1 用户注册

**接口**: `POST /auth/register`

**请求体**：
```json
{
  "username": "zhangsan",
  "password": "Password@123",
  "phone": "13800138000",
  "email": "zhangsan@example.com",
  "healthGoal": "lose_weight",
  "inviteCode": "ABC123" // 可选，邀请码
}
```

**参数说明**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 4-16位字母数字组合 |
| password | string | 是 | 8+位含大小写字母及特殊符号 |
| phone | string | 是 | 11位中国大陆手机号 |
| email | string | 否 | 邮箱地址 |
| healthGoal | string | 是 | lose_weight/gain_muscle/keep_healthy/disease_control/pregnancy |
| inviteCode | string | 否 | 邀请码 |

**响应示例**：
```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "userId": 1001,
    "username": "zhangsan",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 86400
  },
  "timestamp": 1701518400000
}
```

### 2.2 用户登录

**接口**: `POST /auth/login`

**请求体**：
```json
{
  "username": "zhangsan",
  "password": "Password@123",
  "captcha": "1a2b" // 连续失败5次后必填
}
```

**响应示例**：
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
```

## 4. AI功能接口

### 4.1 生成饮食计划

**接口**: `POST /ai/generate-plan`

**请求头**: `Authorization: Bearer <token>`

**请求体**：
```json
{
  "goal": "lose_weight",
  "budget": 100,
  "planDays": 7,
  "allergies": ["花生", "海鲜"],
  "preferences": "不吃辣，喜欢清淡",
  "contextId": "ctx_20241202_001" // 可选，用于多轮对话
}
```

**参数说明**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| goal | string | 是 | lose_weight/gain_muscle/keep_healthy/disease_control/pregnancy |
| budget | number | 是 | 每日预算(元) |
| planDays | number | 是 | 计划天数：3/7/30 |
| allergies | array | 否 | 过敏源列表 |
| preferences | string | 否 | 饮食偏好描述 |
| contextId | string | 否 | 上下文ID，用于多轮对话 |

**响应示例**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "planId": 5001,
    "planName": "7天减脂饮食计划",
    "planDays": 7,

      "dailyCalories": 1450,
      "protein": "85g",
      "carbs": "160g",
      "fat": "45g"
    },
    "shoppingList": [
      { "item": "鸡胸肉", "quantity": "500g", "price": 15 },
```

## 9. 错误码说明

### 9.1 业务错误码

| 错误码 | 说明 | 解决方案 |
|-------|------|---------|
| 40001 | 用户名或密码错误 | 检查登录凭据 |
| 40002 | 用户名已存在 | 更换用户名 |
| 40003 | 手机号已注册 | 使用其他手机号或登录 |
| 40004 | 验证码错误 | 重新获取验证码 |
| 40005 | Token无效或已过期 | 重新登录 |
| 40006 | 权限不足 | 联系管理员 |
| 40301 | AI配额已用完 | 升级会员或明日再试 |
| 40302 | AI服务调用失败 | 稍后重试 |
| 40303 | 上传文件格式不支持 | 检查文件格式 |
| 40304 | 上传文件过大 | 压缩后重新上传 |
| 40401 | 资源不存在 | 检查资源ID |
| 40901 | 请求频率超限 | 降低请求频率 |
| 50001 | 服务器内部错误 | 联系技术支持 |
| 50002 | 数据库异常 | 联系技术支持 |
| 50003 | 第三方服务异常 | 稍后重试 |

## 10. 接口调用示例

### 10.1 JavaScript (Axios)

```javascript
import axios from 'axios'

// 创建axios实例
const api = axios.create({
  baseURL: 'https://api.nutriai.com/api',
  timeout: 30000
})

// 请求拦截器
api.interceptors.request.use(config => {
  const token = localStorage.getItem('nutri_token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  config.headers['X-Request-ID'] = generateUUID()
  return config
})

// 响应拦截器
api.interceptors.response.use(
  response => response.data,
  error => {
    if (error.response?.status === 401) {
      // Token过期，跳转登录
      router.push('/login')
    }
    return Promise.reject(error)
  }
)

// 使用示例
async function generateDietPlan() {
  try {
    const response = await api.post('/ai/generate-plan', {
      goal: 'lose_weight',
      budget: 100,
      planDays: 7
    })
    
    console.log('计划生成成功：', response.data)
  } catch (error) {
    console.error('生成失败：', error.message)
  }
}
```

### 10.2 Java (Spring RestTemplate)

```java
@Service
public class NutriAIClient {
    
    private final RestTemplate restTemplate;
    private final String baseUrl = "https://api.nutriai.com/api";
    
    public DietPlanResponse generateDietPlan(DietPlanRequest request, String token) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.setBearerAuth(token);
        
        HttpEntity<DietPlanRequest> entity = new HttpEntity<>(request, headers);
        
        ResponseEntity<ApiResponse<DietPlanResponse>> response = restTemplate.exchange(
            baseUrl + "/ai/generate-plan",
            HttpMethod.POST,
            entity,
            new ParameterizedTypeReference<ApiResponse<DietPlanResponse>>() {}
        );
        
        return response.getBody().getData();
    }
}
```
