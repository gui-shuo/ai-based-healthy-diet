# Sprint 9 - 前端使用指南

## 🎯 如何在前端页面生成饮食计划

### 方法1: 访问Vue前端页面（推荐）

#### 步骤1: 确保服务运行

```bash
# 终端1: 后端（已启动）
cd backend
java -jar target\nutriai-backend-1.0.0.jar

# 终端2: 前端（已启动）
cd frontend
npm run dev
```

**确认**:
- 后端: http://localhost:8080 ✅
- 前端: http://localhost:3001 ✅

#### 步骤2: 登录系统

1. 打开浏览器访问: http://localhost:3001
2. 点击"登录"按钮
3. 输入账号密码:
   - 用户名: `admin`
   - 密码: `Admin123456`
4. 点击登录

#### 步骤3: 访问饮食计划页面

**方法A**: 通过导航栏
- 登录后，在顶部导航栏找到"AI饮食计划"
- 点击进入

**方法B**: 直接访问
- 浏览器地址栏输入: http://localhost:3001/diet-plan
- 回车访问

#### 步骤4: 生成饮食计划

**快速生成（推荐新手）**:

1. 在页面底部找到"或快速生成"区域
2. 点击以下任一按钮:
   - **7天减脂计划** - 适合减肥人群
   - **7天增肌计划** - 适合增肌人群  
   - **7天健康计划** - 适合维持健康

3. 等待30-60秒，AI正在生成...
4. 生成完成后，页面会显示:
   - ✅ 完整的饮食计划（Markdown格式）
   - ✅ 每日详细菜单
   - ✅ 营养目标
   - ✅ **采购清单**（在计划末尾）
   - ✅ 饮食建议

**完整参数生成（高级用户）**:

1. 填写表单参数:
   - **计划天数**: 7天/14天/30天
   - **目标**: 减脂塑形/增肌强壮/维持健康
   - **运动强度**: 轻度/中度/高强度
   
2. 可选参数（用于精确计算热量）:
   - 性别: 男/女
   - 年龄: 例如 28
   - 身高: 例如 175 cm
   - 体重: 例如 70 kg
   - 每日热量: 留空自动计算
   
3. 饮食偏好:
   - 例如: "喜欢清淡口味"
   - 例如: "不吃辣"
   
4. 过敏/忌口:
   - 例如: "海鲜过敏"
   - 例如: "不吃牛肉"

5. 点击"开始生成饮食计划"按钮
6. 等待生成完成

## 📝 生成的内容包含什么？

### 1. 计划概述
- 计划的特点和预期效果
- 适合人群
- 注意事项

### 2. 营养目标
- 每日总热量
- 蛋白质目标（克数和百分比）
- 碳水化合物目标
- 脂肪目标

### 3. 每日详细计划（7天/14天/30天）
每天包含:
- **早餐** (7:00-8:00)
  - 食材列表
  - 营养成分
  - 小计热量
  - 烹饪提示
  
- **午餐** (12:00-13:00)
  - 食材列表
  - 营养成分
  - 小计热量
  - 烹饪提示
  
- **晚餐** (18:00-19:00)
  - 食材列表
  - 营养成分
  - 小计热量
  - 烹饪提示
  
- **加餐** (可选)
  - 健康零食
  - 营养补充

- **每日营养总计**
  - 总热量
  - 总蛋白质
  - 总碳水
  - 总脂肪

### 4. 采购清单 ✨ **新增！**

按类别分组:
- 🥬 **蔬菜类**: 西兰花、西红柿、黄瓜...
- 🍎 **水果类**: 苹果、香蕉、橙子...
- 🍖 **肉类/蛋白质**: 鸡胸肉、鸡蛋、牛奶...
- 🍚 **主食类**: 大米、燕麦、全麦面包...
- 🥛 **其他**: 调味料、油盐...

每项包含:
- 食材名称
- 所需数量
- 购买单位（斤、个、包等）

### 5. 饮食建议
- 水分摄入建议
- 烹饪方法建议
- 饮食习惯建议
- 运动配合建议

## 🎨 页面功能说明

### 顶部按钮

| 按钮 | 功能 | 说明 |
|------|------|------|
| 导出计划 | 下载Markdown文件 | 只在生成后显示 |
| 重新设置 | 返回参数设置 | 清除当前计划 |

### 表单字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 计划天数 | ✅ | 7/14/30天 |
| 目标 | ✅ | 减脂/增肌/维持 |
| 运动强度 | ❌ | 影响热量计算 |
| 性别 | ❌ | 用于BMR计算 |
| 年龄 | ❌ | 用于BMR计算 |
| 身高 | ❌ | 用于BMR计算 |
| 体重 | ❌ | 用于BMR计算 |
| 每日热量 | ❌ | 留空自动计算 |
| 饮食偏好 | ❌ | 口味偏好 |
| 过敏/忌口 | ❌ | 食物限制 |

### 快捷按钮

| 按钮 | 参数 | 适合人群 |
|------|------|----------|

## ⚠️ 常见问题

### Q1: 为什么没有看到采购清单？

**A**: 采购清单在计划的**末尾**，需要向下滚动查看。在Markdown内容的最后，会有一条分隔线`---`，之后就是采购清单。

### Q2: 生成时间太长怎么办？

**A**: AI生成需要30-60秒，这是正常的。请耐心等待，页面会显示进度条。

### Q3: 生成失败怎么办？

**A**: 可能原因：
- Token过期：重新登录
- 后端未启动：检查8080端口
- AI服务问题：稍后重试

### Q4: 如何修改已生成的计划？

**A**: 
1. 点击"重新设置"按钮
2. 修改参数
3. 重新生成

### Q5: 可以生成多少天的计划？

**A**: 支持7天、14天、30天三种选项。

### Q6: 热量计算准确吗？

**A**: 使用Harris-Benedict公式计算，科学准确。如果提供完整的个人数据（性别、年龄、身高、体重），计算会更精确。
