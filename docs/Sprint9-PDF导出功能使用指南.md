# Sprint 9 - PDF导出功能使用指南

## 🎯 功能概述

PDF导出功能允许用户将生成的饮食计划导出为精美的PDF文档，方便打印和分享。

### 功能特点

- ✅ 一键导出PDF
- ✅ 中文字体支持
- ✅ 精美排版
- ✅ 包含封面页
- ✅ 自动页码
- ✅ 完整内容保留

---

## 📋 使用步骤

### 步骤1: 生成饮食计划

1. 登录系统
2. 进入"AI饮食计划"页面
3. 配置并生成饮食计划
4. 等待生成完成

### 步骤2: 导出PDF

1. 在生成的计划右上角，找到"导出PDF"按钮
2. 点击"导出PDF"按钮
3. 等待PDF生成（通常3-5秒）
4. 浏览器自动下载PDF文件

### 步骤3: 查看PDF

1. 打开下载的PDF文件
2. 查看完整的饮食计划内容
3. 可以打印或分享

## 🔧 技术说明

### 后端技术

- **PDF生成库**：iText 7
- **中文字体**：STSong-Light
- **编码**：UTF-8

### API端点

```
GET /api/diet-plan/export-pdf/{planId}
Headers:
  Authorization: Bearer {token}
Response:
  Content-Type: application/pdf
  Content-Disposition: attachment; filename="计划名称.pdf"
```

### 前端实现

```javascript
// 调用API
const response = await fetch(
  `http://localhost:8080/api/diet-plan/export-pdf/${planId}`,
  {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${token}`
    }
  }
)

// 下载文件
const blob = await response.blob()
const url = URL.createObjectURL(blob)
const a = document.createElement('a')
a.href = url
a.download = `${title}.pdf`
a.click()
```

## 🐛 常见问题

### Q1: 点击"导出PDF"没有反应？

**可能原因**：
1. 后端服务未启动
2. 计划已过期
3. 网络连接问题

**解决方案**：
1. 检查后端是否运行
2. 重新生成计划
3. 检查浏览器控制台错误

### Q2: PDF中文显示乱码？

**原因**：字体文件缺失

**解决方案**：
1. 确保iText font-asian依赖已添加
2. 检查字体配置是否正确

### Q3: PDF文件太大？

**原因**：计划内容过多（如30天计划）

**解决方案**：
- 正常现象，30天计划可能达到1-2MB
- 可以考虑分段导出

### Q4: 提示"此功能仅限黄金会员使用"？

**说明**：会员权限检查已启用

**解决方案**：
- 升级为黄金会员
- 或联系管理员

### Q5: 下载的PDF无法打开？

**可能原因**：
1. PDF生成失败
2. 文件损坏
3. PDF阅读器问题

**解决方案**：
1. 重新导出
2. 尝试不同的PDF阅读器
3. 检查后端日志
