## 1. 设计理念

### 1.1 设计原则

**1. 简洁明了（Simplicity）**
- 界面干净整洁，减少视觉噪音
- 每个页面专注于一个核心任务
- 信息层级清晰，主次分明

**2. 健康活力（Vitality）**
- 使用清新的绿色系为主色调
- 大量留白营造呼吸感
- 圆角设计传递亲和力

**3. 专业可信（Professionalism）**
- 数据可视化清晰准确
- 使用专业的营养学术语
- 提供数据来源和科学依据

**4. 易用性（Usability）**
- 符合用户直觉的交互模式
- 明确的操作反馈
- 容错设计，防止误操作

### 1.2 目标用户画像

**主要用户群体**：
- **年龄**: 25-45岁
- **职业**: 都市白领、健身爱好者、宝妈
- **特征**: 注重健康、有一定消费能力、时间有限
- **痛点**: 不知如何科学搭配饮食、缺乏专业指导

#### 3.1.2 设计细节

**导航栏（Header）**：
- **高度**: 64px
- **背景**: 白色，向下滚动后变为半透明毛玻璃效果
- **Logo**: 左侧，包含图标+文字
- **导航菜单**: 中间，主要功能入口
- **用户信息**: 右侧，头像+昵称+下拉菜单
- **主题切换**: 右上角，日/夜图标

**Hero Banner**：
- **高度**: 500px
- **背景**: 渐变色（primary-400 → primary-600）
- **图片**: 高质量健康食物摄影
- **标题**: 32px Bold，白色
- **描述**: 16px Regular，白色透明度90%
- **CTA按钮**: "开始使用"，白色背景+主色文字
- **切换指示器**: 底部居中，圆点样式
- **自动播放**: 5秒/张，支持手动切换

**功能卡片（Feature Cards）**：
```
卡片尺寸: 280px × 320px
布局: Grid 4列，间距24px
背景: 白色
圆角: 12px
阴影: shadow-md，hover时变为shadow-lg
动画: hover上浮4px + 阴影加深

内容结构:
┌──────────────────────┐
│   🎯 图标 (64×64)    │
│                      │
│   功能标题 (20px)    │
│   功能描述 (14px)    │
│                      │
│   [立即体验] 按钮    │
└──────────────────────┘
```

**统计数据展示**：
- **背景**: 浅绿色渐变
- **数字**: 超大字号（48px），滚动数字动画
- **图表**: 使用ECharts简洁图表
- **布局**: 3列等宽

#### 3.1.3 响应式适配

| 断点 | 宽度 | 布局调整 |
|------|------|---------|
| Desktop | ≥1200px | 4列卡片，完整导航 |
| Tablet | 768-1199px | 2列卡片，收起部分导航 |
| Mobile | <768px | 1列卡片，汉堡菜单 |

## 6. 无障碍设计

### 6.1 WCAG 2.1标准

- **对比度**: 文字对比度≥4.5:1
- **键盘导航**: 所有功能可键盘操作
- **焦点指示**: 明显的焦点样式
- **语义化HTML**: 正确使用标签
- **ARIA属性**: 辅助技术支持

### 6.2 实现要点

```html
<!-- 图片alt属性 -->
<img src="food.jpg" alt="健康的鸡胸肉沙拉">

<!-- 按钮aria-label -->
<button aria-label="关闭对话框">
  <CloseIcon />
</button>

<!-- 表单label关联 -->
<label for="username">用户名</label>
<input id="username" type="text">

<!-- 焦点管理 -->
<div role="dialog" aria-labelledby="dialog-title">
  <h2 id="dialog-title">提示</h2>
</div>
```
