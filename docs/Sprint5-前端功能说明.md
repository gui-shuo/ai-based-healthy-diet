# Sprint 5 - 前端饮食记录页面功能说明

## 📦 已完成的功能

### 1. 核心页面和组件

| 文件 | 类型 | 说明 |
|------|------|------|
| `FoodRecordView.vue` | 页面 | 饮食记录主页面 |
| `NutritionStats.vue` | 组件 | 营养统计图表组件 |
| `FoodRecordList.vue` | 组件 | 饮食记录列表组件 |
| `AddFoodRecordDialog.vue` | 组件 | 添加记录对话框 |

### 2. 功能特性

#### ✅ 营养统计图表（NutritionStats）
- **日期选择器** - 选择查看任意日期的营养数据
- **卡路里总览** - 大卡片显示总卡路里和各餐次分布
- **营养成分卡片** - 4个卡片展示蛋白质、碳水、脂肪、纤维
- **ECharts饼图** - 营养成分占比可视化
- **ECharts柱状图** - 各餐次卡路里分布
- **骨架屏加载** - 优雅的加载状态

#### ✅ 饮食记录列表（FoodRecordList）
- **卡片式布局** - 每条记录一个卡片，含照片和详情
- **照片展示** - 支持照片预览，无照片时显示占位图标
- **快速查看** - 点击卡片查看详情
- **快速删除** - 每条记录可直接删除
- **空状态** - 无记录时显示友好提示
- **响应式设计** - 自适应移动端

#### ✅ 添加记录对话框（AddFoodRecordDialog）
- **餐次选择** - 早餐/午餐/晚餐/加餐单选按钮组
- **食物名称** - 必填，最多100字符
- **记录时间** - 日期时间选择器
- **照片上传** - 集成FoodPhotoUpload组件
- **营养信息** - 6个营养成分输入（份量、卡路里、蛋白质等）
- **备注输入** - 可选，最多500字符
- **表单验证** - 实时验证必填项和格式
- **保存成功** - 自动刷新列表和统计

#### ✅ 食物照片上传（FoodPhotoUpload）
- **点击上传** - 点击区域选择文件
- **拖拽上传** - 支持拖拽图片到上传区域
- **文件验证** - 只允许JPG、PNG、GIF格式，最大10MB
- **上传进度** - 显示上传进度百分比
- **实时预览** - 上传后立即显示照片预览
- **删除功能** - 可删除已上传的照片

#### ✅ 记录详情对话框（FoodRecordDetailDialog）
- **照片展示** - 大图展示食物照片
- **基本信息** - 描述表格展示所有字段
- **营养成分** - 分区展示营养数据
- **备注显示** - 卡片式展示备注信息

#### ✅ 筛选和分页
- **日期范围筛选** - 选择开始和结束日期
- **餐次类型筛选** - 按餐次类型过滤
- **分页展示** - 支持10/20/50/100条每页
- **跳转功能** - 可直接跳转到指定页码

### 5. API服务

**文件位置**: `src/services/foodRecord.js`

**提供的方法**:
- `createFoodRecord(data)` - 创建饮食记录
- `getFoodRecords(params)` - 查询饮食记录（分页）
- `getFoodRecordById(id)` - 获取记录详情
- `deleteFoodRecord(id)` - 删除记录
- `getNutritionStats(date)` - 获取营养统计
- `uploadFoodPhoto(file)` - 上传食物照片

**辅助方法**:
- `getMealTypeList()` - 获取餐次类型列表
- `getMealTypeByValue(value)` - 根据value获取餐次类型
- `MealTypes` - 餐次类型枚举对象

## 📱 使用流程

### 添加饮食记录
1. 点击"添加记录"按钮
2. 选择餐次类型（早/午/晚/加餐）
3. 输入食物名称
4. 选择记录时间
5. （可选）上传食物照片
6. 输入营养信息
7. （可选）添加备注
8. 点击"保存"

### 查看统计
1. 页面顶部自动显示今日统计
2. 点击日期选择器查看其他日期
3. 查看卡路里总览和各餐次分布
4. 查看营养成分饼图和柱状图

### 筛选记录
1. 使用日期范围选择器筛选时间段
2. 使用餐次类型下拉框筛选餐次
3. 列表自动更新显示筛选结果

### 查看详情
1. 点击任意记录卡片
2. 弹出详情对话框
3. 查看完整信息和大图

### 删除记录
1. 点击记录卡片上的"删除"按钮
2. 确认删除操作
3. 列表和统计自动更新
