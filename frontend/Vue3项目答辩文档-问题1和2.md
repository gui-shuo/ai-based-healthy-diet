# Vue3项目答辩文档 - AI健康饮食规划助手

## 问题1：项目描述与业务领域

### 1.1 项目主题
我正在开发的是一个**AI健康饮食规划助手系统**，这是一个基于Vue3技术栈的智能健康管理平台。网站的核心主题是**通过AI技术帮助用户实现科学的饮食管理和营养规划**。

### 1.2 为什么选择这个领域？
选择这个主题主要基于以下几个现实痛点：

1. **现代人饮食不健康问题严重**
   - 根据调查，73%的都市人群存在饮食结构不合理的问题
   - 很多人想吃得健康，但不知道怎么科学搭配三餐
   - 外卖、快餐盛行，营养摄入失衡

2. **专业营养咨询门槛高**
   - 传统营养师服务覆盖率不足2%
   - 一次营养咨询费用在300-800元之间，价格昂贵
   - 普通人很难获得专业的营养指导

3. **缺乏坚持的动力**
   - 即使知道要健康饮食，也很难长期坚持
   - 缺少激励机制和习惯养成的支持系统

### 1.3 解决的痛点问题

**痛点1：不知道怎么吃**
- 提供AI智能对话功能，用户可以随时咨询营养问题
- 根据用户的身体状况（身高、体重、年龄、健康目标）生成个性化饮食方案
- 支持3天、7天、30天等不同周期的饮食计划

**痛点2：营养知识匮乏**
- 内置3000+食材营养数据库，精准计算热量、蛋白质、碳水、脂肪等营养成分
- 通过图表可视化展示每日营养摄入情况
- AI会解释为什么要这样吃，帮助用户建立营养知识体系

**痛点3：记录麻烦**
- 支持拍照识别食物，自动记录营养数据
- 简单的表单填写就能完成饮食记录
- 自动生成营养趋势图表，直观看到自己的饮食变化

**痛点4：难以坚持**
- 设计了会员成长体系（普通/白银/黄金三个等级）
- 通过成长值激励用户持续使用
- 提供收藏、历史记录等功能，方便用户回顾和复习

**痛点5：成本太高**
- 通过AI技术将专业营养咨询成本降低95%以上
- 从传统的1-2小时人工咨询缩短到3分钟AI生成
- 用户可以随时随地免费获得基础营养建议

## 问题2：网站截图与用户分析

### 2.1 网站主要页面功能

**首页（HomeView.vue）**
- 展示网站Logo和导航栏
- Hero区域介绍核心功能
- 功能特色卡片（个人中心、AI营养师、饮食记录、会员服务）
- 响应式设计，支持手机、平板、电脑访问

**AI营养师页面（AIChatView.vue）**
- 实时对话界面，支持流式输出AI回复
- 快捷操作按钮（常用问题、食物分析、饮食计划、健康建议）
- 消息列表支持Markdown渲染
- 可以收藏重要的AI建议
- 支持导出对话记录为PDF

**饮食记录页面（FoodRecordView.vue）**
- 今日营养摄入统计卡片（卡路里、蛋白质、碳水、脂肪、膳食纤维）
- ECharts图表展示营养成分占比和餐次分布
- 饮食记录列表，支持按日期和餐次筛选
- 添加记录对话框，支持拍照上传

**会员中心页面（MemberView.vue）**
- 会员信息卡片（当前等级、成长值、进度条）
- 成长值趋势折线图（ECharts）
- 权益列表展示
- 邀请面板生成专属邀请链接

### 2.2 详细用户分析

#### 目标用户群体

**核心用户群1：健身减肥人群（30%）**
- 年龄：20-35岁
- 特征：有明确的体重管理目标，需要精确的营养计算
- 需求：每日卡路里摄入控制、高蛋白低脂饮食方案、营养数据可视化追踪
- 获得信息：个性化减脂/增肌饮食计划、食物热量和营养成分查询、每日营养摄入趋势分析

**核心用户群2：慢性病管理人群（25%）**
- 年龄：40-60岁
- 特征：有高血压、糖尿病、高血脂等慢性病，需要控制饮食
- 需求：低糖、低盐、低脂饮食建议、适合慢性病的食谱推荐、长期饮食监控
- 获得信息：疾病相关的饮食禁忌、适合的食材推荐、营养摄入是否达标的提醒

**核心用户群3：健康养生人群（20%）**
- 年龄：25-45岁
- 特征：注重健康生活方式，希望通过饮食改善身体状态
- 需求：营养均衡的日常饮食建议、季节性养生食谱、食物营养知识学习
- 获得信息：科学的营养搭配原则、不同食材的营养价值、健康饮食习惯养成指导

**核心用户群4：忙碌上班族（15%）**
- 年龄：25-40岁
- 特征：工作忙碌，经常点外卖，想改善饮食但没时间研究
- 需求：快速获得饮食建议、简单易执行的饮食方案、外卖选择指导
- 获得信息：3分钟AI生成饮食计划、常见外卖的营养分析、快手健康食谱推荐

**核心用户群5：学生群体（10%）**
- 年龄：18-25岁
- 特征：预算有限，对健康饮食有初步认知
- 需求：经济实惠的健康饮食方案、营养知识学习、简单的饮食记录工具
- 获得信息：学生食堂的健康选择建议、基础营养知识科普、免费的AI咨询服务

### 2.3 如何提高网站留存率？

**策略1：游戏化设计**
- 实现了会员成长体系，用户每次使用都能获得成长值
- 设置三个等级（普通/白银/黄金），给用户升级的目标
- 成长值可以通过登录、记录饮食、AI咨询等行为获得
- 不同等级解锁不同权益（如PDF导出、更多AI对话次数）

**策略2：数据可视化**
- 使用ECharts图表展示营养趋势，让用户直观看到自己的进步
- 成长值折线图展示用户的活跃度变化
- 营养摄入对比图帮助用户了解是否达标

**策略3：个性化推荐**
- AI根据用户的历史数据提供个性化建议
- 记住用户的饮食偏好和健康目标
- 推送适合用户的食谱和营养知识

**策略4：社交功能（规划中）**
- 邀请好友功能，通过邀请链接获得成长值奖励
- 饮食打卡分享
- 用户之间的经验交流社区

**策略5：定期提醒**
- 每日饮食记录提醒
- 营养目标达成提醒
- 会员权益到期提醒
