# NutriAI 营养知识库

## 权威数据来源

本知识库数据来源于《中国食物成分表标准版（第6版）》，由中国疾病预防控制中心营养与健康所编著，
数据获取自中国营养学会官方平台 (https://nlc.chinanutri.cn/fq/)。
本系统共收录 **1354** 种真实食物的详细营养成分数据，覆盖谷薯类、蔬菜、水果、畜禽肉、水产、蛋奶、豆类、坚果等全品类。
所有营养数据均以每100g可食部分计。

辅助参考：
- 中国居民膳食指南（2022版）—— 中国营养学会
- 中国居民膳食营养素参考摄入量（DRIs 2023版）
- 国家食品安全风险评估中心数据

- 第五层：盐<5g，油25-30g

## 五、特殊人群营养建议

### 减重人群
- 能量缺口：每日减少300-500kcal
- 蛋白质：1.2-1.6g/kg体重（维持肌肉量）
- 推荐食物：鸡胸肉(133kcal)、鱼类(低脂高蛋白)、蔬菜(低热量高纤维)
- 避免：油炸食品、含糖饮料、高脂肪零食

### 增肌人群
- 能量盈余：每日增加300-500kcal
- 蛋白质：1.6-2.2g/kg体重
- 推荐食物：鸡蛋(143kcal, 蛋白质13.3g)、牛肉(瘦)(106kcal, 蛋白质20.2g)、牛奶
- 碳水化合物：训练前后补充复合碳水（燕麦、红薯、糙米）

### 孕期营养
- 孕早期：叶酸600μgDFE/天，碘230μg/天
- 孕中晚期：能量+300kcal/天，蛋白质+15g/天
- 钙：1000mg/天（推荐牛奶、豆腐、芝麻酱）
- DHA：200mg/天（推荐深海鱼）
- 铁：29mg/天（推荐红肉、动物肝脏）

### 老年人营养
- 蛋白质：1.0-1.2g/kg体重
- 钙：1000mg/天 + 维生素D 15μg/天
- 推荐：奶制品、豆制品、深色蔬菜、鱼类
- 少量多餐，注意咀嚼能力适应的食物形态

### 糖尿病饮食
- 优选低GI食物（GI≤55）：燕麦、全麦面包、豆类、大部分蔬菜
- 膳食纤维：25-30g/天
- 控制碳水化合物总量，均匀分配到三餐
- 限制精制糖、白米饭、甜饮料

### 高血压饮食（DASH原则）
- 钠：<2000mg/天（约5g盐）
- 钾：3500-4700mg/天（推荐香蕉、土豆、菠菜）
- 增加蔬果、低脂乳制品、全谷物
- 限制腌制食品、加工肉类
